\doxysubsection{Detailed Description}
Represents a game scene. 

The \doxylink{class_scene}{Scene} class is an abstract base class that represents a game scene. It provides methods to load, input, update, render, and unload the scene. It also provides methods to calculate delta time, begin rendering, end rendering, and get the next scene.
