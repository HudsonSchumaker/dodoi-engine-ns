\doxysubsection{Detailed Description}
Manages the lifecycle of game scenes. 

The \doxylink{class_scene_manager}{Scene\+Manager} class is responsible for managing the lifecycle of game scenes. It provides methods to add scenes, load scenes asynchronously, and get the instance of the \doxylink{class_scene_manager}{Scene\+Manager}. It uses the Singleton design pattern.
